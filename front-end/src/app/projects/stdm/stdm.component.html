Project Description
  • C++ application that reads from a file input packets to a multiplexer and computes the output speed and packet distribution in frames
  • Input files contain how many channels feed the multiplexer, how many packets arrive on a given channel, and what time those packets arrive
  • Packet arrival and output frame diagrams are printed to the screen along with average packet transmission speed
  • Run on an Ubuntu (linux) virtual machine
