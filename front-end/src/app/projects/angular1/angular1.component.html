Project Description
  • This project is a full stack single page eBay search application that utilizes the eBay search API. The backend server is hosted on the Google Cloud Platform (GCP) running NodeJS.
  • The frontend is also hosted on GCP and was built with Angular. It queries the backend server asynchronously which then queries eBay. Additionally the frontend utilizes the bootstrap library such that the app is responsive.
  • The data for the wishlist items is stored on MongoDB.
